GPM Monitor - Introduction

The GPS Positioning Module Monitor provides a graphical user interface for Magna's RTK GPS position tracking system.

Communication

Connect to GPM module via one of the listed communication ports. If the module supports a WiFi console, then it's IP address will be listed in the drop-down list.

Ethernet Server

The Ethernet server feature can create a bridge between a USB Virtual COM port from the GPM module and a network Ethernet port, as if it is an additional WiFi console. This will allow GPM monitor to run across the network, and communicate with the GPM module, without requiring access to the WiFi interface of the module.

Simulink

The Simulink feature provides an interface for the embedded Simulink model in the GPM module, to use the Ethernet interface for communicating with external applications, using the module's USB Virtual COM port.

The IP address to the external application must be provided.

Status Control

Displays GPM module system time, and the power supply voltage. The slide can be used to adjust the update rate.

Static

Various fields can be monitored.

Some controls are provided to interface with the emedded Simulink model.

The checkbox "NTRIP" can be used to activate or deactivate the link with the RTK correction data server (CORS).
